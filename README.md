# PixelMC Welcome

A lightweight Forge 1.20.1 / Java 17 server-side welcome message mod for PixelMC / Vanilla Era.

Current status: repository bootstrap only. Welcome message features are not implemented yet.

## Environment

- Minecraft: 1.20.1
- Forge: 47.3.33
- Java: 17
- Mod ID: `pixelmcwelcome`

## Current Scope

This stage contains only a minimal loadable Forge mod class and build configuration. It does not implement any welcome message behavior.

## Planned Features

- Configurable welcome messages
- Placeholders
- Color, HEX, and gradient parsing
- Login count tracking
- Total playtime tracking
